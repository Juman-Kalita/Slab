Problem: Still shows 0 inventory
Solution: 
  - Make sure you clicked "RUN" in Supabase
  - Go to http://localhost:5173/setup again
  - Click "Initialize Database" again

Problem: Error in Supabase SQL Editor
Solution:
  - Make sure you copied the ENTIRE supabase-setup-clean.sql file
  - Try refreshing Supabase page and running again

Problem: Can't access /setup page
Solution:
  - Make sure dev server is running (npm run dev)
  - Try: http://localhost:5173/setup (with /setup at end)
